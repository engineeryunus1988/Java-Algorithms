## 🛠️ Teknik Bilgi Notlarım

### 🧠 Stack (Yığın) Veri Yapısı
`Valid Parentheses` sorusunda kullandığım **Stack** yapısı, **LIFO (Last In First Out)** prensibiyle çalışır. Bu yapı; derleyici (compiler) tasarımı, geri alma (undo) mekanizmaları ve derinlik öncelikli aramalarda (DFS) kritik rol oynar.

### 🔄 String ve Karakter İşleme
`Roman to Integer` çözümünde uyguladığım tersten tarama tekniği, karmaşık kuralları daha basit ve okunabilir bir algoritmaya dönüştürmeyi sağlamıştır.
